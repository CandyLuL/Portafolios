Mis Proyectos

Mi Restaurante Demo

Una **aplicación web interactiva** diseñada para simular un sistema completo de pedidos de comida online. Los usuarios pueden explorar un menú con **productos ficticios**, utilizar un **sistema de búsqueda** avanzado, añadir productos a un **carrito de compras dinámico** y visualizar los "puntos del restaurante". Ideal para negocios de comida que buscan una plataforma eficiente y fácil de usar.

Dulce Tentación

Una atractiva **página web de venta de dulces** diseñada para ofrecer una experiencia de compra deliciosa. Incluye un **carrito de compras funcional**, un catálogo de **productos bien presentados con imágenes**, y una sección de **reseñas de clientes** para generar confianza. Perfecta para cualquier negocio que desee vender productos dulces de forma online.

Guía de Estudio Interactiva

Un **recurso online interactivo** creado para facilitar el aprendizaje y la preparación académica. Ofrece una **guía con diferentes materias**, un menú intuitivo para acceder a cada tema, y **dos simulacros de examen de opción múltiple**. Los usuarios pueden revisar sus respuestas, ver sus aciertos y reiniciar las pruebas, haciendo el estudio más eficiente y dinámico.
